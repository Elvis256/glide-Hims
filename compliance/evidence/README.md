# Compliance Evidence Repository
This directory stores auditable evidence for security, SDLC, vulnerability management, operations, and customer assurance controls.
## Layout
- `templates/`: standard evidence templates.
- `records/YYYY-MM/`: completed monthly evidence records.
- `releases/`: release-specific evidence packs.
## Operating Rule
Templates are immutable baselines. Completed records are append-only and must include date, owner, and verification status.
